OSM Pub Guide

The OSM Pub Guide will help you to find the right pub in Münster.

Depending what you want to do you can search for different attributes. Use the search box to start your query. Clicking at the different buttons at the bottom you can select what kinds of pubs should be seen on the map. If you would like to get more information then you can use the introductional guide: Start guide

The OSM Pub Guide is a project of the Institude of Geoinformatics (ifgi).
